# open-data-ai-analytics

## Опис та мета

Метою проєкту є аналіз відкритого набору даних про правила проведення спортивних змагань: виконання оцінки якості даних (структура, типи полів, пропуски, дублікати) та подальше дослідження закономірностей оновлення документів, їх актуальності й розподілу між видами спорту із застосуванням Python (Pandas) і засобів візуалізації.
---

## Джерело даних
Набір відкритих даних: **Правила змагань з видів спорту**

Ресурс:  
https://data.gov.ua/dataset/017ce04c-5a83-4832-b116-2b9bafca5cfa/resource/1208ea90-47e7-46a0-a5ad-b91c15bd3cd6/download/list_11-09-2025.csv

### Поля датасету
- `identifier` — ідентифікатор запису
- `sportName` — вид спорту
- `type` — тип документа
- `publisherName` — орган, що затвердив
- `publisherIdentifier` — код органу
- `title` — назва документа
- `dateAccepted` — дата прийняття
- `number` — номер наказу
- `status` — чинність документа
- `url` — посилання на документ

## Питання для аналізу та гіпотези

### 1. Зв’язок популярності спорту та кількості правил
**Гіпотеза:** популярні (зокрема олімпійські) види спорту мають більше затверджених і оновлених правил змагань, ніж менш поширені.

Що аналізується:
- кількість документів на кожен вид спорту
- частота оновлення правил

### 2. Динаміка затвердження правил з часом
**Гіпотеза:** у новіші роки кількість затверджених правил змагань більша, ніж у попередні періоди.

Що аналізується:
- кількість наказів за роками
- періоди найбільш активного оновлення

### 3. Актуальність нормативних документів
**Гіпотеза:** значна частина правил змагань оновлюється або втрачає чинність протягом кількох років після затвердження.

Що аналізується:
- статус документів (чинний / нечинний)
- середній строк дії правил
